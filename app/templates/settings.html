Setup Guide

Quick notes

Home Assistant
  1. Create a webhook-triggered automation in Home Assistant.
  2. Copy the generated webhook URL.
  3. Paste it into Webhook Endpoint URL.
  4. Use Save + Send Test Notifications to confirm delivery.
Discord
  1. Create an incoming webhook in the target Discord channel.
  2. Paste the URL into Discord Webhook URL.
  3. Save, then send a test notification.
OIDC
  1. Set Public Base URL to the external address users will visit.
  2. Create an OIDC client in Authelia with callback {{ (app_settings.app_base_url.rstrip('/') if app_settings.app_base_url else 'https://your-app.example.com') }}/auth/callback.
  3. Fill in the issuer URL, client credentials, and group mapping values here.
  4. Save, then sign out and back in to validate the login flow.
